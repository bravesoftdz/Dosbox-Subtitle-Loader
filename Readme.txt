hid.dll положить в папку с dosbox.exe
Файлы .subs положить в папку GRAPHIC\FST\

Работает только с "output=surface" и "output=direct3d"
Субтитры к файлу C005.FST, который я переименовал у себя в VIRGLOGO.FST
Запись WAV делается для всех файлов, к которым есть соответствующие .subs файлы(Даже пустые). Отключается в SubsPlugin.ini, параметр DoWavRecord.


Шрифт: "Times New Roman",H:20,W:16,C:002FC12F,B:00000000,F:BIT,O:-80

"Times New Roman", - Название шрифта
H:20,       - Высота 
W:16,       - Ширина
C:002FC12F, - Цвет текста
B:00000000, - Цвет Фона
F:BIT,      - B=Bold, I=Italic, T=Прозрачный фон
O:-80       - Смещение по вертикали, Положительное=От верхней границы, Отрицательное=От нижней границы
